# SwasthAI Frontend

This frontend is a `Vite + React + React Router + TypeScript` app.

It is not a Next.js app.

## Scripts

```bash
npm install
npm run dev
npm run build
npm test
```

## Current MVP scope

- landing page
- chat UI
- text symptom input
- live call to backend `POST /analyze`
- emergency alert banner

Not connected yet:

- voice transcription
- image analysis
- hospital map
- patient summary card
- audio responses
